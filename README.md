ATTENDANCE TRACKER PROJECT 
This project is a Bash shell script that automates the setup of an Attendance Tracker Project. The script creates the directory structure and files, allows the user to configure attendance thresholds, and check whether Python 3 is installed. The project structure contains an attendance_checker.py file, a Helpers directory for configuration and a reports directory for attendance reports.

The script also includes error handling and process management. If the user interrupts exucation using Ctrl+C, a backup archive is created and the incomplete project directory is removed before the scripts exits. 

To run the project: ./setup_project.sh
